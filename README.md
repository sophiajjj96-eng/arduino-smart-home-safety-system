# RAD Smart Home Safety Device

Prototype smart home safety system designed to detect fire and flood risks and automatically shut off connected devices.

## Overview

This project was developed as part of the Rapid Assembly and Design Challenge (RAD).  
The system monitors temperature and water levels using sensors connected to an Arduino microcontroller.

When a risk threshold is detected, the system automatically shuts off connected devices to prevent potential hazards.

## Key Features

- Temperature monitoring using a thermistor
- Water level detection using a water sensor
- Automated device shutdown when thresholds are exceeded
- Arduino-based control system
- CAD enclosure design with 3D printing

## System Components

- Arduino microcontroller
- Thermistor temperature sensor
- Water level sensor
- Breadboard and wiring
- LED indicator
- Water pump
- 3D printed enclosure
